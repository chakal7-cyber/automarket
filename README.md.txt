AutoMarket - Plateforme de Location et Vente de Véhicules

AutoMarket est une application web complète de gestion de location et vente de véhicules d'occasion. Développée avec PHP, MySQL, HTML5, CSS3 et JavaScript, elle offre une expérience utilisateur fluide pour la recherche, la réservation et l'achat de véhicules.
Fonctionnalités

Gestion des utilisateurs
- Inscription et connexion sécurisées
- Gestion de session avec PHP
- Rôle utilisateur / administrateur

Catalogue de véhicules
- Affichage des véhicules à vendre et à louer
- Filtres dynamiques (marque, prix, carburant, transmission, places)
- Tri des résultats (prix croissant/décroissant, année)
- Recherche en temps réel via API

Réservation de véhicules
- Formulaire de réservation avec calcul automatique du prix
- Options supplémentaires (GPS, siège bébé, assurance premium)
- Confirmation par email
- Gestion des disponibilités

 Contact et demandes de devis
- Formulaire de contact
- Demandes de devis pour les véhicules
- Notification par email

 Interface utilisateur
- Design moderne et responsive
- Animations CSS fluides
- Menu mobile adaptatif
- Images dynamiques (Unsplash / personnalisables)
